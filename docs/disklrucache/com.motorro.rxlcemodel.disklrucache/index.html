Creates DiskLRU caching delegate for SyncDelegateCacheService Delegate uses cache directory provided by DiskLruCacheProvider. This directory is designed to be shared between several delegates. Thus we need to provide each delegate an unique DiskLruCacheSyncDelegate.prefix to not to mix data with other delegates. The DiskLruCacheSyncDelegate.sd is a serializer/deserializer that saves/restores entity from file streams.
